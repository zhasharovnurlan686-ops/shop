# Интернет-магазин — README

Современный адаптивный интернет-магазин на чистом HTML, CSS и JavaScript.  
Поддерживает добавление товаров в корзину и сохранение через LocalStorage.

---

# 🚀 Возможности

- Адаптивный дизайн
- Каталог товаров
- Корзина
- Подсчет общей суммы
- Сохранение корзины в LocalStorage
- Hover-анимации
- Современный UI
- Работает без backend
